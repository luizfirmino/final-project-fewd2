Demo for using Modal Window

Use Modal Window when you need to give, for example, additional info. When user clicks on link, Modal Window will pop up with that info.

1. Include .css file for "Modal Window"
2. Add a href with id "openModal" into HTML document: <a href="#openModal">Open Modal</a>.
3. Add div tag to hold all items of modal window construction into HTML document: <div id="openModal" class="modalWindow"> <div> <a href="#" class="close">×</a> <h2>Modal Window</h2> <p>This is a sample modal window.</p> </div> </div>.
